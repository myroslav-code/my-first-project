# training.py
Программа для управления пользователями, их инвентарём и ролями (создатель, администратор, пользователь).

## Возможности
- Регистрация и вход пользователей
- Система ролей (создатель, админ, пользователь)
- Управление инвентарём
- Бан/разбан пользователей
- Повышение уровня
- Добавление/удаление пользователей (для админов/создателя)

## Файл данных
Все данные пользователей хранятся в `data.json` (создаётся автоматически).
